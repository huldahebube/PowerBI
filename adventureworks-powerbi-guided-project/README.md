## Project Type
Guided / Learning Project (Power BI)

## Overview
This project is a guided Power BI dashboard built using the AdventureWorks dataset. 
The goal of the project was to strengthen hands-on skills in data modeling, DAX, 
and dashboard design while following industry-style reporting requirements.

The dashboard focuses on analyzing sales performance across time, geography, 
and product attributes.

## Key Features
- KPI cards showing:
  - Total Sales Amount
  - Total Quantity
  - Total Freight
  - Period-over-Period % Change
- Monthly sales trend analysis
- Sales distribution by state/province (map visual)
- Sales breakdown by product color
- Interactive filtering by year
- Detailed tabular view with sales, freight, quantity, and YTD metrics

## Skills Demonstrated
- Power BI dashboard development
- Data modeling and relationships
- DAX measures and calculated metrics
- Time-based analysis (YTD, trends)
- Data visualization and layout design
- Business-focused reporting
